# VoiceDraw AI 语音绘图助手

## 项目简介

VoiceDraw 是一个轻量级 AI 语音/文字绘图工具。用户可以输入绘图指令，例如“在左边画一个红色圆形”“在右边画一个蓝色矩形”，系统会识别指令中的颜色、图形和位置，并在画布中自动完成绘制。

## 当前已实现功能

- 支持文字指令输入
- 支持绘制圆形和矩形
- 支持识别红色、蓝色、绿色、黑色、黄色、紫色
- 支持识别左边、右边、中间、上方、下方位置
- 支持撤销上一步
- 支持清空画布
