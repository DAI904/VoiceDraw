# VoiceDraw AI 语音绘图助手

## 一、项目简介

VoiceDraw 是一个基于语音指令的轻量级 AI 绘图工具原型。用户可以通过文字输入或语音输入绘图指令，例如“在左边画一个红色圆形”“在右边画一个蓝色矩形”，系统会识别指令中的图形、颜色和位置，并在画布中自动完成绘制。

本项目选择题目二“AI 语音绘图工具”，重点实现“语音输入 → 指令解析 → Canvas 绘图”的核心交互流程。

## 二、项目功能

目前已实现以下功能：

- 支持文字指令输入
- 支持语音识别输入
- 支持绘制圆形和矩形
- 支持识别红色、蓝色、绿色、黑色、黄色、紫色
- 支持识别左边、右边、中间、上方、下方位置
- 支持撤销上一步
- 支持清空画布
- 支持浏览器 Canvas 绘图展示

## 三、使用方法

1. 使用 Chrome 或 Edge 浏览器打开 `index.html`。
2. 可以在输入框中输入绘图指令。
3. 点击“执行指令”按钮后，系统会根据指令自动绘图。
4. 也可以点击“开始语音识别”按钮，通过语音输入绘图指令。
5. 如果浏览器弹出麦克风权限提示，请选择“允许”。
6. 可以点击“撤销”按钮撤销上一步操作。
7. 可以点击“清空画布”按钮清空当前画布。

## 四、支持的指令示例

### 1. 基础绘图指令

- 在左边画一个红色圆形
- 在右边画一个蓝色矩形
- 在中间画一个绿色圆形
- 在上方画一个黄色圆形
- 在下方画一个紫色矩形

### 2. 操作类指令

- 撤销
- 返回上一步
- 清空画布

## 五、技术栈

本项目主要使用以下技术：

- HTML：负责页面结构
- CSS：负责页面样式
- JavaScript：负责交互逻辑、语音识别、指令解析和绘图功能
- Canvas：负责图形绘制
- Web Speech API：负责浏览器语音识别能力

## 六、核心实现思路

系统整体流程如下：

1. 用户输入文字指令，或点击“开始语音识别”按钮说出绘图指令。
2. 浏览器将语音内容识别为文字。
3. JavaScript 对文字内容进行关键词解析。
4. 系统根据关键词判断颜色、图形和位置。
5. 调用 Canvas API 在画布上绘制对应图形。
