# VoiceDraw AI 纯语音绘图助手设计文档

## 一、项目背景

本项目对应题目二“AI 语音绘图工具”，目标是开发一款纯语音控制的绘图工具。用户在完成浏览器麦克风授权后，不需要通过鼠标或键盘进行绘图创作，而是通过语音指令完成图形绘制、组合图案生成、复杂指令执行、撤销、清空和保存图片等操作。

题目要求重点关注以下几个方面：

1. 指令理解的准确性与容错性；
2. 语音到绘图操作的响应延迟；
3. 复杂指令的拆解与执行能力；
4. 实现应用的同时，需要提交设计文档，记录计划支持的指令能力、最终实现情况以及未完成部分的原因说明。

基于以上要求，本项目采用前端网页方式实现，使用 HTML、CSS、JavaScript、Canvas API 和浏览器 Web Speech API 构建一个轻量级 AI 语音绘图原型系统。

---

## 二、项目目标

本项目希望实现一个简单、直观、低成本的语音绘图工具，使用户可以通过自然语言语音指令完成绘图操作。

项目主要目标包括：

1. 支持用户通过语音输入绘图指令；
2. 支持浏览器将语音转换为文本；
3. 支持从文本指令中解析图形类型、颜色、位置和操作意图；
4. 支持绘制基础图形和组合图案；
5. 支持生成一幅完整的随机风景画；
6. 支持复杂语音指令的拆解和顺序执行；
7. 支持撤销、清空画布、保存图片等画布操作；
8. 显示指令解析与 Canvas 绘图执行耗时，用于体现响应延迟；
9. 保持系统轻量化，尽量不依赖后端服务和额外付费接口。

---

## 三、纯语音控制说明

由于浏览器安全策略限制，网页首次调用麦克风时必须由用户主动点击按钮并授权。因此，本项目页面保留了“启动语音控制”和“停止语音控制”按钮。

需要说明的是：

- 启动按钮仅用于触发浏览器麦克风授权；
- 麦克风授权完成后，绘图创作过程通过语音完成；
- 用户可以通过语音完成绘图、清空、撤销、保存和停止语音识别；
- 项目的核心交互方式是语音控制，而不是键盘输入或鼠标绘制。

## 四、计划支持的指令能力

项目设计初期，计划支持以下几类语音指令能力：

### 1. 基础图形绘制指令

计划支持用户通过语音绘制基础图形，例如：

- 画一个圆形
- 画一个矩形
- 画一个三角形

### 2. 颜色控制指令

计划支持用户通过语音指定图形颜色，例如：

- 红色圆形
- 蓝色矩形
- 黄色三角形

### 3. 位置控制指令

计划支持用户通过语音指定图形位置，例如：

- 在左边画一个圆形
- 在右边画一个矩形
- 在中间画一个三角形
- 在上方画一个太阳
- 在下方画一个图形

### 4. 组合图案绘制指令

计划支持一些由多个 Canvas 图形组合而成的图案，例如：

- 太阳
- 房子
- 云朵
- 星星
- 笑脸

### 5. 场景生成指令

计划支持用户通过一句语音生成完整画面，例如：

- 画一幅风景画
- 生成一幅完整画
- 画一个场景

### 6. 画布操作指令

计划支持用户通过语音完成画布管理操作，例如：

- 清空画布
- 撤销
- 返回上一步
- 保存图片
- 停止语音控制

### 7. 复杂指令拆解能力

计划支持用户一次说出包含多个步骤的复杂指令，例如：

- 清空画布，然后画一幅风景画，再在左边画一个太阳
- 画一个太阳，然后在右边画一朵云
- 画一幅风景画，接着在上方画一颗星星

系统需要能够识别连接词，并将复杂指令拆解为多条简单指令后依次执行。

## 五、最终实现的功能

经过开发，本项目最终实现了以下功能。

### 1. 纯语音控制绘图

用户点击“启动语音控制”完成麦克风授权后，可以直接说出绘图指令。系统会调用浏览器 Web Speech API 进行语音识别，并将识别结果传入指令解析模块。

已实现示例：

- 在左边画一个红色圆形
- 在右边画一个蓝色矩形
- 在中间画一个黄色三角形
- 画一个太阳
- 在右边画一朵云
- 在中间画一座房子

### 2. 基础图形绘制

已实现以下基础图形：

- 圆形
- 矩形
- 三角形

系统会根据用户指令中的关键词判断需要绘制的图形类型。

### 3. 颜色识别

已实现以下颜色识别：

- 红色
- 蓝色
- 绿色
- 黑色
- 黄色
- 紫色
- 橙色
- 粉色

例如，用户说“在左边画一个红色圆形”，系统会识别“红色”为颜色，“圆形”为图形，并在左侧绘制红色圆形。

### 4. 位置识别

已实现以下位置识别：

- 左边
- 右边
- 中间
- 上方
- 下方

例如，用户说“在右边画一个蓝色矩形”，系统会将绘图位置映射到画布右侧区域。

### 5. 组合图案绘制

已实现以下组合图案：

- 太阳
- 房子
- 云朵
- 星星
- 笑脸

这些图案不是单一图形，而是由多个 Canvas 基础绘图操作组合生成。例如，太阳由圆形和光线组成，房子由矩形、三角形、门和窗户组成。

### 6. 随机风景画生成

已实现语音生成完整风景画功能。

用户可以说：

- 画一幅风景画
- 生成一幅完整画
- 画一个场景

系统会自动绘制包含多个元素的风景画，例如：

- 天空
- 草地
- 太阳
- 云朵
- 远山
- 房子
- 树木
- 花朵
- 小路
- 河流等自然元素

其中部分元素会带有随机变化，例如天空颜色、草地颜色、太阳位置、云朵数量、花朵颜色和位置等，使每次生成的画面略有不同。
